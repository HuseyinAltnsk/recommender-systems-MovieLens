critics is a user-item matrix used in a recommender system and consists of two files:

critics_ratings.data
This is tab-delimited file with the following string fields:
(1) user name
(2) item ID
(3) user rating
(4) 0 (a placeholder)

critics_movies.item
This file maps item ID's to movie titles and consists of the following string fields:
(1) item ID
(2) | (separator)
(3) movie title
